## Установка FFmpeg
Что бы все работало корректно, необходимо установить FFmpeg

1. Скачайте FFmpeg для Windows: https://www.gyan.dev/ffmpeg/builds/ffmpeg-release-full.7z
2. Распакуйте и скопируйте `ffmpeg.exe` в папку `backend/bin/`
3. Убедитесь что файл называется `ffmpeg.exe`

## Запуск приложения
Фронтенд запускается:
1. Перейти в директорию frontend: cd frontend
2. Выполнить команду: npm run dev
3. Фронтенд запустится на порту 8002
4. Фронтенд доступен по http://localhost:3000/ или http://192.168.0.69:3000/ или http://172.27.128.1:3000/

Бэкенд:
1. Запуск производится из корня проекта(...\emoji_for_video): uvicorn backend.app.main:app --port 8002 --reload
2. Бэкенд на http://127.0.0.1:8002 - он снаружи не доступен, доступен только фронтенд.
